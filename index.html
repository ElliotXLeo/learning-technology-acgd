¿Qué aprenderás en este curso?

Todo lo necesario y esencial de React, (Create React App, JSX, Components, Props, State, Conditional Rendering, Basic Hooks, Styled Components y más)

JSX

Extensión de la sintaxis de JavaScript. Recomendamos usarlo con React para describir cómo debería ser la interfaz de usuario.

Components

Es un elemento de software visual que tiene su propio estado, recibe unas propiedades e implementa su propia lógica de renderizado.

Styled Components

Alternativa de mejorar CSS para diseñar componentes en React
